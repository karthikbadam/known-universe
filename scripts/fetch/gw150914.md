# GW150914 strain — fetch instructions

## Source

> Abbott, B. P. et al. (LIGO Scientific & Virgo Collaborations) (2016).
> *Observation of Gravitational Waves from a Binary Black Hole Merger*.
> PRL 116, 061102.
> [doi:10.1103/PhysRevLett.116.061102](https://doi.org/10.1103/PhysRevLett.116.061102)

Real strain data is hosted on the Gravitational Wave Open Science
Center (GWOSC):

```sh
# H1 strain, 4096 Hz, 32 s around GW150914:
curl -fL -O \
  https://losc.ligo.org/s/events/GW150914/H-H1_LOSC_4_V2-1126259446-32.hdf5
```

Native format is HDF5; convert to CSV with `gwpy` or `h5py` and
downsample to ~512 Hz for shipping. Window around the merger (t_c ≈ 16 s
in GPS seconds in the file) and re-center to t = 0.42 s to match our
plot's time axis.

## Target schema

```
t_s,strain,strain_model
```

with strain in units of 10⁻²¹ (multiplied for plot legibility) and a
6-line `#` provenance header.

## Flip the UI

const dataStatus: DataStatus = "simulated";  // → "real"
